# Программа для накрутки рефералов в проекте Pentagon Games

# Принцип работы

Программа открывает сайт по реферальной ссылке и вводит данные. 
После этого ждет, когда вы пройдете recaptcha и зарегестрируете аккаут.
После того как вас перекидывает на окно успешной регистрации, то окно закрывается
и открывается новое

# Установака библиотек

```
pip install selenium
```
```
pip install fake-useragent
```
```
pip install webdriver-manager
```
# Запуст программы

В текстовый документ помещяете данные в виде:
```
username*mail*password*ref
```
Пример:
```
Dave999*Dave999@rambler.ru*qwerty*https://pentagon.games/sign-up/liza34
```
Потом запускаете программу# Pentagon-Games
